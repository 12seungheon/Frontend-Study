### 숫자 타입

```jsx
var integer = 10; //정수
var double = 10.12; // 실수
var negative = -20; // 음의 정수
```

### 문자열 타입

```jsx
var string;
string = "문자열"; // 작은따옴표
string = "문자열"; // 큰따옴표
string = `문자열`; // 백틱(ES6)
```

### 템플릿 리터럴

```jsx
var template = `Template literal`;
console.log(template); // Template literal
```

### 불리언 타입

```jsx
var foo = true;
console.log(foo); // true

foo = false;
console.log(foo); // false
```
